Definición y tipos de conjuntos

⏱ 45 min

¿Qué es un conjunto?

Un conjunto es una colección bien definida de objetos distintos, llamados elementos o miembros del conjunto. La teoría de conjuntos es la base de toda la matemática moderna y nos permite organizar y clasificar objetos con propiedades comunes.

Definición formal

Un conjunto A es una colección de objetos bien determinados y distintos entre sí. Si un objeto x pertenece al conjunto A, se escribe x ∈ A; si no pertenece, se escribe x ∉ A.

Los conjuntos pueden representarse de tres formas: por extensión (listando sus elementos), por comprensión (describiendo una propiedad) o mediante diagramas de Venn.

Ejemplo

El conjunto de los números pares menores que 10 puede escribirse:
• Por extensión: A = {2, 4, 6, 8}
• Por comprensión: A = {x ∈ ℕ | x es par y x < 10}

🏷️
Tipos de conjuntos

Existen diferentes clasificaciones de conjuntos según sus características y el número de elementos que contienen.

Conjunto vacío

No contiene ningún elemento. Se denota como ∅ o {}. Ej: el conjunto de números pares impares.

Conjunto finito

Tiene un número determinado de elementos. Ej: A = {1, 2, 3, 4, 5} tiene cardinalidad 5.

Conjunto infinito

Sus elementos no pueden contarse completamente. Ej: ℕ = {1, 2, 3, ...}

Conjunto universal

Contiene todos los elementos del contexto estudiado, se denota como U o Ω.

Subconjunto

El conjunto A es subconjunto de B si todo elemento de A también está en B. Se escribe A ⊆ B.

Conjuntos iguales

Dos conjuntos son iguales si tienen exactamente los mismos elementos: A = B ↔ A ⊆ B y B ⊆ A.

Implementación en MATLAB

Aprende cómo representar y manipular conjuntos usando MATLAB. Sigue los pasos en orden y ejecuta el código en tu entorno.

1

Definir conjuntos en MATLAB

En MATLAB los conjuntos se representan como vectores o arreglos. Usa corchetes para definirlos:

% Definir conjuntos como vectores
A = [2, 4, 6, 8];
B = [1, 2, 3, 4, 5];

% Mostrar los conjuntos
disp('Conjunto A:'); disp(A);
disp('Conjunto B:'); disp(B);
💡 Tip

MATLAB no tiene un tipo de dato "conjunto" nativo, pero los vectores con la función unique() eliminan duplicados automáticamente.

2

Verificar pertenencia de un elemento

Para verificar si un elemento x ∈ A, usa la función ismember():

A = [2, 4, 6, 8];

% ¿El 4 pertenece a A?
if ismember(4, A)
  disp('4 ∈ A: Verdadero');
else
  disp('4 ∉ A: Falso');
end
3

Cardinalidad y conjunto vacío

La cardinalidad (número de elementos) se obtiene con length() o numel():

A = [2, 4, 6, 8];
vacio = [];  % Conjunto vacío

fprintf('|A| = %d\n', length(A));
fprintf('|∅| = %d\n', length(vacio));

% Verificar si es vacío
if isempty(vacio)
  disp('El conjunto es vacío');
end
✅ Resultado esperado

Al ejecutar este código deberías ver: |A| = 4 y |∅| = 0 en la ventana de comandos de MATLAB.
